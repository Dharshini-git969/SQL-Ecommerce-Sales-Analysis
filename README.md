# Ecommerce Sales Analysis using SQL

## Project Overview

This project demonstrates SQL-based data analysis on an ecommerce sales dataset using MySQL. The objective is to analyze sales performance, revenue generation, product popularity, customer ratings, and business trends through structured SQL queries.

The project focuses on transforming raw transactional data into meaningful business insights that support data-driven decision-making. Various SQL concepts such as aggregation, grouping, filtering, ranking, conditional statements, and window functions have been used to answer real-world business questions.


# Objectives

The primary objectives of this project are:

- Analyze total sales revenue.
- Identify top-selling products.
- Evaluate revenue generated across different product categories.
- Compare city-wise sales performance.
- Analyze average customer ratings by category.
- Rank products based on sales performance.
- Classify products into sales categories.
- Demonstrate practical SQL skills for business analytics.
